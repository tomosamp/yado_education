# 受け入れ基準書

## 1. 認証
- [ ] 招待URLから初回設定できる
- [ ] ログイン/ログアウトが正常動作する
- [ ] 無効ユーザーは利用不可

## 2. 学習機能
- [ ] 受講者がコース/レッスンを閲覧できる
- [ ] レッスン詳細でPDF・ヒント・提出状況を確認できる
- [ ] ヒントは段階開示で動作する

## 3. 提出・レビュー
- [ ] 自動判定課題は判定OKコードのみ提出可能
- [ ] Webアプリ講座でGitHub URL提出が可能
- [ ] レビューコメント必須で承認/差戻しできる
- [ ] 差戻し後は再提出で新規審査になる

## 4. 進捗
- [ ] 合格のみ完了として集計される
- [ ] ダッシュボードに主要指標が表示される

## 5. 管理機能
- [ ] 管理者がコース/レッスン/ユーザー/ラベルを管理できる
- [ ] 招待メール送信ができる

## 6. 非機能
- [ ] Docker構成でローカル起動できる
- [ ] テスト (`php artisan test`, `npm run test`, `npm run test:e2e`) が成功する
- [ ] `docs` に日本語名Markdown文書が揃っている
