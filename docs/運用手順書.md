# 運用手順書

## 1. 日次運用
- Mailpitで招待メール/再設定メールを確認。
- レビュー待ち件数を `レビュー待ち一覧` で処理。
- 失敗判定ログ（`judge_runs.status=error|timeout`）を監視。

## 2. 起動・停止
```bash
cd <このリポジトリのルート>/infra
docker compose up -d

docker compose down
```

## 3. DBバックアップ（例）
```bash
docker exec yado-mysql mysqldump -uyado -psecret yado_education > backup.sql
```

## 4. MinIOバックアップ（例）
- バケット `yado-education` を定期的に同期。
- 例: `mc mirror local/yado-education /backup/minio/yado-education`

## 5. トラブルシュート
- 判定失敗: `backend` / `queue-worker` コンテナログ確認。
- PDF表示不可: MinIO接続設定（`AWS_ENDPOINT`, `AWS_BUCKET`）確認。
- ログイン不可: `SANCTUM_STATEFUL_DOMAINS` と CORS 設定確認。
