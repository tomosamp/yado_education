# 画面仕様書

## 1. 受講者
- `/login`: ログイン
- `/dashboard`: 進捗ダッシュボード
- `/curriculum`: コース/レッスン一覧
- `/sections/:sectionId`: レッスン詳細、PDF表示、ヒント、提出
- `/mypage`: 表示名・パスワード変更、提出履歴

## 2. レビュー
- `/reviewer/pending`: レビュー待ちユーザー一覧
- `/reviewer/pending/users/:userId`: 指定ユーザーのレビュー待ちレッスン一覧
- `/reviewer/pending/submissions/:submissionId`: 提出詳細レビュー（提出コード/提出情報表示、合格/差戻し）

## 3. 管理者
- `/admin/dashboard`: 全体状況、招待送信
- `/admin/categories`: コース管理
- `/admin/sections`: レッスン管理
- `/admin/users`: ユーザー管理（モーダル招待、受講者招待時のラベル選択）
- `/admin/labels`: ラベル管理

## 4. 共通仕様
- 左サイドバーはロール別にメニューを制御。
- APIエラーはトースト通知で表示。
- 画面はSP/PCの両方で崩れないレスポンシブ構成。
